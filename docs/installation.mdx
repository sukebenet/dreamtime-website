## Requirements

- 64-bit operating system:
  - Windows 7 or superior.
  - [Compatible Linux distribution](https://snapcraft.io/docs/installing-snapd).
  - macOS Catalina or superior.

### CPU

- **12 GB** of RAM.

### GPU (Optional)

- **NVIDIA GPU** (AMD GPU's are not supported)
- Minimum [3.5 CUDA compute capability](https://developer.nvidia.com/cuda-gpus). (GeForce GTX 780+)
- [Latest NVIDIA drivers](https://www.nvidia.com/Download/index.aspx).
- **6 GB** of GPU VRAM.
- **8 GB** of RAM.

:::caution
NVIDIA RTX 3000 series GPU's are not yet compatible with DreamTime.
:::
