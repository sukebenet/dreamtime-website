DreamTime by itself is just a user interface that makes the whole nudification process more accessible, but for everything to work it is necessary to install external components:

- **DreamPower:** Command line application with the nudification algorithms.
- **Checkpoints:** DeepNude pre-trained models required by DreamPower.
- **Waifu2X:** Optional command line application with the image upscaling and denoise algorithm.

On this page we offer download links and manual installation instructions.

### Installation

- Inside DreamTime click on the button `DreamTime Folder` to open the `AppData` folder of DreamTime.
- Create a folder called `dreampower`, if it exists delete it and create it again.
- Extract the contents of the 7z file (you can use [7-Zip](https://www.7-zip.org/)) into the `dreampower` folder you just created.
